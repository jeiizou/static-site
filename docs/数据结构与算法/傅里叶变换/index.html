傅里叶变换

简单来说, 傅里叶变换就是把一个输入信号分解成一堆正弦波的叠加.

利用傅里叶变化, 我们可以把音频中最重要的成分表达出来, 并且得到和原始声音非常接近的波形. 在计算机中, 用一系列数据点的形式来存储.
